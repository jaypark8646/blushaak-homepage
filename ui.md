# Blu Shaak COFFEE - UI 설계 문서

## 1. 디자인 시스템

### 1.1 브랜드 컬러
| 용도 | 컬러 | 비고 |
|---|---|---|
| Primary | 블루 계열 | 로고, 주요 강조 요소 |
| Accent | 민트/틸 | 선택 상태, 토글 활성 |
| CTA (Action) | 레드 | 문의하기, 그랜드 오픈 등 강조 버튼 |
| Background | 화이트 / 라이트 그레이 | 본문 배경 |
| Footer | 다크 그레이 / 블랙 | 푸터 영역 배경 |
| Text | 블랙 / 다크 그레이 | 본문 텍스트 |

### 1.2 브랜드 무드
- **키워드:** 바다, 파도, 휴양지, 도심 속 휴식, 자연
- **톤앤매너:** 고급스러운 감성 + 편안한 리조트 무드
- **이미지 스타일:** 바다/자연 배경 + 커피/베이커리 + 따뜻하면서도 시원한 톤

### 1.3 로고
- **심볼:** 파란색 상어 지느러미/파도 형태
- **텍스트:** "Blu Shaak COFFEE"
- **위치:** GNB 중앙
